V2 API Information

The V2 API uses a new architecture with improved interfaces and dependency injection.

Available Endpoints:

  • /v2/stream/video-feed - Stream video with license plate detection
  • /v2/stream/snapshot - Get a single frame from the camera
  • /v2/detection/detect-from-camera - Detect license plate from camera
  • /v2/results/detections - Get all license plate detections
  • /v2/results/enhanced-results - Get all enhanced results
  • /v2/results/statistics - Get detection statistics
